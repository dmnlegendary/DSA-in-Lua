# 🐢 DSA in Lua

Welcome to **DSA-in-Lua** — a playful yet practical repository where I explore **Data Structures and Algorithms (DSA)** using the Lua programming language.  
This project is part of my journey as an AI Engineering student, blending technical rigor with creative flair.

---

## ✨ What’s Inside?
- 📂 **src/** → Core Lua implementations (arrays, stacks, queues, linked lists, trees, etc.)
- 📂 **examples/** → Demo scripts to test and showcase each structure
- 📂 **docs/** → Notes, explanations, and learning resources

---

## 🎯 Goals
- Practice **pattern matching, table manipulation, and validation routines** in Lua.
- Build a reusable library of DSA implementations for learning and freelancing.
- Document everything clearly so others can follow along.
